# 第一章 成事有道：达成目标的时间管理

1. 成事目标：没有目标，则不用学时间管理。我有目标，我想赚更多钱，想实现财富自由，但这不是目标，是“念想”，每个人都有无数“念想”。
2. 成事动机：只有改变，人生才有更多的可能。只有改变，才能改变当前和过去的情况，未来才能变化
3. 成事智慧：均衡的人生，应该有五个维度的目标：工作事业、婚姻家庭、个人财富、生活健康、学习成长

# 第二章 理解时间：管理过去 、现在和未来

1. 回击过去：如果不改变过去的自己，绝不可能扭转现状
2. 干预现在：你的未来已来，不过不是你想要的样子，把控现在的变化趋势才能达成未来的自己
3. 活在未来：弱者活在过去，强者活在未来。努力过好现在是一种痛苦的幸福

# 第三章 时间押注：能否成事的关键

1. 学会押注时间，获得更高的回报，学会将时间押注在什么的人、事上
    - 什么人：任何时候都选择大概率上能让我们变得更好的人，尽可能多靠近比自己更优秀的人，向他们学习
    - 什么事：每个阶段都要把时间押注在最重要的事情上，押注在让自己变得更好的事情上
2. 明确时间押注方向，一流世界管理的前提是明确目标
3. 时间押注聚焦，无法专注的人，注定一无所成(成吉思汗曾说，他一天只干3件事情：吃饭、睡觉、打仗)
4. 时间押注密度，绝对性的投入，大力出奇迹大力，快速崛起。在每一个阶段、每一个核心事情上押注，进行压倒性投入
5. 时间押注监控，每个阶段都重新确认时间押注

# 第四章 时间定义：厘清重点，拒绝瞎忙

1. 人生中四类事：重要且紧急的事、重要但不紧急的事、紧急但不重要的事、不重要且不紧急的事；动作性的、紧迫性的、短暂性的、问题性的事
2. 尊从做事惯性：人为何喜欢做更紧急而非更重要的事，这是人性导致。
3. 学会事件转换：坚定做重要但不紧急的事，转化时整体要遵循以下原则：
    - 原则一：将弹性的转化为限时的。设定一个具体的、限时的目标
    - 原则二：将长期的转化为短期的。拆分目标实现，列出具体任务清单
    - 原则三：将抽象的转化为具体的。拆解任务，列出具体的待办清单
4. 学会留白：合理减少重要且紧急的事情
5. 阻断无意义：大量减少不重要且紧急的事
6. 做时间智者：管理不属于自己的世界，很多时间不是完全属于自己的。但公平的是，我们也可以使用别人的时间
    - 管理不属于自己的时间
    - 管理他人的时间：提前预约、提前明确告知占用时长、善于沟通争取别人的时间
    - 管理者如何管理别人的时间：
        - 合理评估目标任务，制定推进时间表
        - 统一讲解，分配任务落实得到人(一定要统一讲解，不要分别进行。因为项目需要大家协作完成。实际执行的时候，很多时候在某个节点上，需要第一个人做完，第二个人才能跟上，或者需要几个人共同配合，一起完成)
        - 确认任务和时间，达成共识
        - 把控进度，保障任务按时完成(按时接收汇报、主动了解情况)



​	我们的执行力多用在重要且紧急的事上，可能只能做到优秀，如果想要从优秀跨越到卓越，就要把更多的执行力用在重要但不紧急的事上。

# 第五章 计划执行：坚决践行，过可控人生

做计划->计划分解->日计划载体->日记计划制定->日计划执行->应对不可控

时间管理的前提是明确目标
1. 做计划：没有计划清单，就没有时间管理，也就是目标。
    - 时间管理的前提是明确目标。计划清单是为目标服务的，是从目标倒推出来的。时间的流动方向比时间的流逝速度更能决定人生，而目标决定了时间的流动方向。这是宏观的、长周期的时间管理。
    - 任何事情都是先在头脑中构思，也就是智力上的第一次创造，然后再付诸实践，也就是体力上的第二次创造。在第一次和第二次创造之间，还有第1.5次创造，那就是把头脑中的第一次创造落实到纸面上，形成计划清单，让创造可视化。
2. 计划分解：日、周、月、年计划
3. 日计划载体：每次做事清单的存在价值
4. 日计划制订：制作每日做事清单的五大要点
5. 日计划执行：执行每日做事情清单的三大原则
    - 按优先级顺序执行，不必完全今日事今日毕
    - 接受不完美，不完美的执行好过没有规划
    - 执行必须标记，记录必须诚实，复盘必须坚持
6. 应对不可控：面对新事情该如何应对
    - 如果我们每天要做的事都是提前规划好的，那么我们在时间管理上的烦恼会少一半，但生活的常态是，我们一边要处理每日做事清单上的待办事项，一边要应付不断冒出来的新事情
    - 时间管理的一条原则是，永远不要火急火燎地行动，而要先判断，再做事。面对任何一件新事情，我们只有四种应对方法——“两做”和“两不做”，分别是马上做、填充进每日做事清单待做，以及转给别人做和拒绝做
    - 【马上做】评估可以马上完成，在三五分钟内完成。这样的事你能快速搞定，基本不会影响你正在处理的事，同时，这些事延后做可能会花费你制作每日做事清单或写备忘便签的时间，或者这件事没做会让你一直挂念着，从而消耗你的精力，所以这些事要马上做。
    - 【填充进每日做事清单待做】一旦一件事要花费10分钟以上的时间去处理，且对方不需要你即时处理，那么最好先将其填充进每日做事清单，挑选合适的时间再做
    - 不做，可以给合适的人做、授权给别人做、拒绝

# 第六章 省时高效：做好时间分配，提高时间效能

1. 提高效率，做到如下4点，时间利用率提升2倍
    - 早睡早起，让黄金时间效率爆发
    - 快速切换，事来则应，事去则静
    - 一气呵成，提高一次性完成率
    - 并联做事，一个时间段推进多件事

2. 减少浪费：坚持这四点，每天少浪费2小时
    - 时刻提醒，减少闲谈
    - 立即行动，减少神游
    - 调节时间，错峰做事
    - 碎片时间，充分利用

3. 节省时间：学会这三招，每天节省2小时
    - 机械工作，速战速决
    - 实现标准化、流程化、可复用。真正的勤奋，是凡事都设计更好的流程和标准，打造流畅的操作系统

4. 不断寻找更优方案
    - 不要因为一件事过去总是这样做，就觉得一定要这样做，要经常思考有没有更优的方案——效果同样好甚至更好，但可以节省大量时间的方案

5. 时间买手：成为时间买手，学会付费外包。
    - 不值得花时间的事情。如果你有不错的赚钱能力
    - 不擅长的事情
    - 不喜欢的事情
    - 升级服务就可以提高效率的事情
    - 学习成长的事情。最好的进步方式就是站在高手的肩膀上学习。不管你想提升哪个领域的哪项能力，基本上很多高手都愿意收费让你踩肩膀，你要做的就是拿着预算找合适的肩膀。

# 第七章 告别顽疾：消除顽疾，成为时间管理高手

1. 拖延：有拖延症怎么办
2. 执行力：执行力差怎么办
    - 什么是执行力：很简单，执行力，就是把想法变成行动，拿到结果的能力
    - 不要还没实施就否定，可以先进行数据分析，基于数据做决策、可行性
    - 先定小目标，然后迭代；凡事列计划，尽管你不必严格遵循计划。列计划，无须很细致，也无须很靠谱。你只需要列一个大概的计划就行，规划你先做什么，再做什么，然后做什么……需要注意的是，一定要给每件事确定一个大致的时间节点，即什么时间一定要开始做完什么
3. 作息：怎么早睡早起
4. 效率：做事效率低怎么办
    - 提高做事效率，能带来巨大的回报
    - 列出当天要做的事，并划分时段
    - 开始做事时，确定准确的截止时间
    - 对任何一件事情有流程感和紧凑的推进意识
    - 尽可能屏蔽外界干扰，让自己当下专心，做一件事时尽量避免被打扰，心里不要挂念其他事情
    - 合理安排更多事项，宁可忙也不要闲，即人潜意识会在事情少情况下放松，放松流程感和推进意识
5. 情绪：不因情绪导致无法高效做事，做好情绪控制
6. 娱乐：沉迷娱乐怎么处理
7. 完美：过度追求完美怎么办

# 第八章 重塑时间管理观：在有限的人生，成最大的事

1. 精力保证：这四份时间，永远不能吝啬，分别是睡眠、吃饭、学习、亲情时间
2. 选择很重要：这三件事情，是一生中最浪费时间的，分别是投入一段不合适的感情、坚持一份不合适的工作、在错误目标上持续投入
3. 重塑时间观：未来常迎，当时可杂，过往不忘
    - 同人的世界观、人生观、价值观一样重要，一个人的时间观
    - 拥有良好的时间观，才能更好的稳定有序、高效做事

时间管理，并不能真正解决"做完事情"这个问题，应该追求的是，制定合理适量的计划，有序可控地推进执行
